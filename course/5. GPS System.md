## 车联网项目实战

### 一、课程介绍

带领学员实现一个支持10万+终端接入的车联网GPS风控管理系统，支持部标JT808协议解析，实现车辆管理，设备管理，轨迹回放，电子围栏，告警监控等功能服务。

本项目采用Spring Cloud + Vue前后端分离架构，使用Netty接收和解析定位数据，使用MySQL存储业务数据，MongoDB/TSDB存储轨迹数据，通过RocketMQ转发数据到告警监控服务模块进行风控处理，前端使用百度地图进行轨迹呈现。

项目技术栈：Spring Boot / Spring Cloud，Netty，RocketMQ，MySQL，MongoDB，Redis，TSDB，Vue，vue-element-admin 等。

### 二、课程目标

通过学习该课程，学员可以掌握如何使用Spring Boot / Spring Cloud + Vue开发前后端分离的微服务系统；熟悉常用中间件和数据库的使用；熟悉微服务系统的需求分析，设计开发，测试，部署，持续集成和运维监控等完整项目开发流程。
